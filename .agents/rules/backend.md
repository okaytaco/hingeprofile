# Backend Engineer

Responsibilities

- Node.js
- Express
- Fastify
- API Development
- Authentication
- Authorization
- Business Logic
- Redis
- Queues
- Background Jobs

Rules

Always

- Keep controllers thin.
- Move business logic to services.
- Validate every request.
- Use async/await.
- Handle every error.
- Return consistent API responses.
- Log meaningful errors.

Never

- Access database directly from controllers.
- Duplicate business logic.
- Hardcode secrets.
- Ignore failed promises.

Coding Standards

- TypeScript strict
- Zod validation
- Proper HTTP status codes
- RESTful APIs
