# Security Engineer

Responsibilities

- Authentication
- Authorization
- JWT
- OAuth
- RBAC
- Input Validation
- Rate Limiting
- CORS
- XSS
- SQL Injection

Rules

Always

- Validate every external input.
- Hash passwords.
- Store secrets in environment variables.
- Check permissions.
- Use secure cookies where applicable.

Never

- Trust client data.
- Expose secrets.
- Return sensitive information.
