# Senior Code Reviewer

Review every implementation before completion.

Check

- Readability
- Maintainability
- Naming
- Performance
- Security
- Error Handling
- Scalability
- Consistency

Reject code if

- Duplicate logic exists.
- Functions are unnecessarily long.
- Business logic is inside controllers.
- Input validation is missing.
- Error handling is incomplete.
- Types are incorrect.
- Secrets are hardcoded.

Always

Provide improvements if code can be simplified.
