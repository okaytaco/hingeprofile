# Code Review Workflow

## Objective

Review code quality without modifying behavior unless requested.

## Steps

1. Understand the purpose of the code.
2. Review architecture.
3. Review correctness.
4. Review readability.
5. Review maintainability.
6. Review performance.
7. Review security.
8. Identify code smells.
9. Suggest improvements.
10. Summarize findings by priority.
